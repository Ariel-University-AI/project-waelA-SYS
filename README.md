## תיאור הפרויקט

**מנת"ח חקלאי** היא מערכת ניתוח תכנוני אוטומטית המיועדת למתכננים, אדריכלים ופקידי רישוי העוסקים בהיתרי בנייה למבנים חקלאיים. המערכת מנתחת תיקי בקשה (תוכניות, גרמושקה, חוות דעת אגרונום ועוד) ומפיקה חוות דעת מקצועית מובנית — תוך שימוש ב-AI מתקדם (Google Gemini).

### מה המערכת עושה?
- **OCR חכם:** מחלצת טקסט וטבלאות מ-PDF בעזרת `pdfplumber`, ומנתחת גם שרטוטים ותמונות
- **ניתוח AI:** שולחת את כל המסמכים ל-Google Gemini לניתוח מעמיק לפי כללי משרד החקלאות
- **ניטור real-time:** מציגה בזמן אמת אילו סעיפים כבר נוצרו (📋 סיכום / ☑️ בדיקת סף / 🚫 חסמים וכו')
- **פלט מובנה:** מחזירה חוות דעת ב-10 לשוניות: סטטוס, מסמכים, בדיקת סף, מפה נורמטיבית, ניתוח, קביעות, השוואה, חסמים, חוות דעת, נספח
- **ייצוא JSON:** ניתן להוריד את כל הניתוח כקובץ JSON

## ארכיטקטורה טכנית

```
מסמכים (PDF/JPG/PNG/DOCX)
        ↓
pdfplumber OCR  ──→  טקסט + טבלאות
        +
קובץ גולמי (base64)
        ↓
Google Gemini 3.1 Pro (streaming)
        ↓
JSON מובנה → 10 לשוניות תצוגה
```

| רכיב | טכנולוגיה |
|------|-----------|

| AI Model | Google Gemini 3.1 Pro Preview |

| ניהול קוד | GitHub |

## הרצה מקומית

### דרישות מוקדמות
- Python 3.10+
- מפתח Google Gemini API (מ-[Google AI Studio](https://aistudio.google.com/))

### שלבים
```bash
# שכפול הריפו
git clone https://github.com/Ariel-University-AI/project-waelA-SYS.git
cd project-waelA-SYS

# התקנת תלויות
pip install -r requirements.txt

# הגדרת מפתח API (אופציונלי — ניתן גם דרך ה-UI)
set GOOGLE_API_KEY=AIza...

# הרצה
streamlit run מנתח_חקלאי_streamlit.py
```
גישה: [http://localhost:8501](http://localhost:8501)

## פריסה על Streamlit Cloud

1. גש ל-[share.streamlit.io](https://share.streamlit.io) והתחבר עם GitHub
2. **New app** → בחר ריפו `project-waelA-SYS` → קובץ: `מנתח_חקלאי_streamlit.py`
3. תחת **Advanced settings → Secrets** הוסף:
   ```toml
   GOOGLE_API_KEY = "AIza..."
   ```
4. לחץ **Deploy** — לאחר כ-2 דקות האפליקציה פעילה בכתובת ציבורית קבועה
